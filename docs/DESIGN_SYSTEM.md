# Design System de Aelius

## Emoción principal

Aelius debe transmitir calma, claridad y confianza.

No busca presionar al usuario. Busca ayudarlo a bajar el ruido mental y ordenar sus pensamientos.

## Principios visuales

- Calma antes que urgencia.
- Espacio antes que saturación.
- Claridad antes que decoración.
- Colores suaves antes que alertas agresivas.
- Lenguaje humano antes que lenguaje técnico.

## Paleta inicial

| Uso | Color | Propósito |
|---|---|---|
| Fondo principal | `#F8FBF9` | Blanco cálido y descansado |
| Verde Aelius | `#4F8F77` | Calma, foco y naturaleza |
| Azul agua | `#CDEBE3` | Serenidad y suavidad |
| Texto principal | `#1F2933` | Lectura clara |
| Texto secundario | `#64756D` | Información de apoyo |
| Bordes | `#DCE9E3` | Separación suave |
| Tarjetas | `#FFFFFF` | Contenido limpio |
| Atención suave | `#D97A6E` | Requiere atención sin generar ansiedad |
| Espera | `#E6B95C` | Pendiente o por resolver |

## Lenguaje

Aelius debe hablar como un compañero sereno.

Ejemplos:

- Enviar → Compartir
- Hoy → Mi día
- Ideas → Inspiración
- Reflexiones → Diario
- Tu foco ahora → Lo más importante hoy

## Frase guía

Aquí puedo bajar el ritmo, ordenar mis pensamientos y volver a enfocarme en lo que realmente importa.
