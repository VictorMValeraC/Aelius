# Aelius

Aelius es un asistente personal de productividad, reflexión y priorización.

No nace como una simple aplicación de tareas, sino como una arquitectura a proyectar: un compañero digital diseñado para ayudar a transformar ideas, pendientes y decisiones en acciones claras.

## Propósito

Ayudar a organizar la mente, priorizar mejor, ejecutar con foco y aprender en el camino.

## Estado del proyecto

Fase 0: Documentación fundacional.

## Componentes previstos

- Frontend: React + Vite
- Hosting: Netlify
- Base de datos: Supabase
- Autenticación: Supabase Auth
- Repositorio: GitHub

## Principios

- Simple antes que complejo
- Seguro desde el inicio
- Conversacional antes que burocrático
- Escalable sin perder humanidad
- Aprender construyendo
