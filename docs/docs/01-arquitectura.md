# Arquitectura a proyectar

## Arquitectura V1

Usuario
→ Aplicación Web
→ Supabase Auth

## Componentes

- React + Vite para la interfaz
- Supabase para autenticación y base de datos
- Netlify para despliegue
- GitHub para control de versiones

## Principio clave

La primera versión debe ser pequeña, clara y funcional.
